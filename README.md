# UtangLink

Digitizing sari-sari store utang transactions using Stellar.

## Problem

Neighborhood stores in the Philippines still track informal debt using paper notebooks, causing unpaid balances and disputes.

## Solution

UtangLink records debt agreements on Soroban smart contracts and allows customers to repay instantly using Stellar wallets.
